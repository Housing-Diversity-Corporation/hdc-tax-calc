### Conformity Status Definitions

| Status | Meaning |
|--------|---------|
| Rolling | Automatically adopts federal changes as enacted |
| Static - Current | Conforms to IRC as of a recent date |
| Static - Lagged | Conforms to an older IRC version |
| Selective | Cherry-picks specific provisions |
| n.a. | No state income tax |

## §1400Z-2 OZ Conformity (Novogradac November 2025)

### States That DO NOT Conform

| Code | Name | Status | Notes |
|------|------|--------|-------|
| CA | California | none | Static conformity to pre-TCJA IRC (Jan 1, 2015) |
| MA | Massachusetts | none | Individuals do not conform; corporate only |
| NC | North Carolina | none | Explicitly decoupled; requires add-back |
| NY | New York | none | Decoupled April 2022, retroactive to Jan 1, 2021 |

### States With PARTIAL Conformity (In-State Only)

| Code | Name | Status | Notes |
|------|------|--------|-------|
| AR | Arkansas | partial | In-state QOZs only (SB196, Feb 2019) |
| HI | Hawaii | partial | In-state QOZs only (SB1130, June 2019) |
| MS | Mississippi | partial | In-state QOZs only; fully aligned Jan 1, 2025 |

### All Other States
Full OZ conformity through rolling or static conformity to IRC post-TCJA.
